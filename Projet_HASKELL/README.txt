Voici notre projet LOGICAL MASTER.
Les fichiers implémentés se trouvent dans"\Projet\Logical\app"
Pour le lancer:
- ouvrir un terminal dans le dossier app
- entrer la commande: ghc -o main main.hs
- Entrer la commande: ./main

Un autre main avec des valeurs différentes ainsi que deux fichier testant ont été
implémentés dans le dossier test. Pour les faire fonctionner, Veuillez les déplacer
dans le dossier app.
